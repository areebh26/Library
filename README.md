 ## ‚ú® Features

- **Add New Books:** Users can input book details (Title, Author, Pages, Read Status) via a dynamic form.
- **View Library:** Displays all books in a responsive grid/table layout.
- **Toggle Read Status:** Interactive button to switch a book's status between "Read" and "Not Read" directly on the book card.
- **Remove Books:** Option to delete specific books from the library collection.
- **Input Validation:** Prevents empty or invalid entries from being added to the library.

## üß† Concepts Applied

This project focuses on the following JavaScript concepts:
1.  **Object Constructors:** Storing book data within initialized objects.
2.  **Prototypes:** Adding methods (like toggling read status) to the Book prototype to save memory.
3.  **Arrays:** Managing the collection of Book objects.
4.  **Looping:** Iterating through the library array to render the DOM elements.
5.  **Data Attributes:** Using `data-index` or unique IDs to associate DOM elements with their corresponding logic objects.
